## 시작하기

### 프로그램 실행

1. 바탕화면의 "금일작업현황 관리" 아이콘을 더블클릭합니다.
2. 프로그램 창이 열립니다 (약 2-3초 소요).

### 로그인

1. 이름을 입력합니다.
2. "로그인" 버튼을 클릭하거나 Enter 키를 누릅니다.

> 💡 **팁**: 이름은 활동 로그에 기록되므로 정확히 입력하세요.

## 기본 사용법

### 1. 작업 입력하기

#### 날짜 선택
- 화면 상단의 날짜를 클릭하여 원하는 날짜를 선택합니다.
- ◀ / ▶ 버튼으로 날짜를 이동할 수 있습니다.

#### 데이터 입력

**순번**: 자동으로 표시됩니다 (1-10).

**계약번호**: 
- 형식: `SH-YYYY-NNN-T`
- 예시: `SH-2026-001-T`
- 자동으로 대문자로 변환됩니다.

**선사**: 
- 고객사 이름 입력
- 예시: `KURILSKIY RYB`

**선명**: 
- 작업 선박 이름
- 예시: `RUT`

**엔진모델**: 
- 메이커 + 모델명
- 예시: `CAT' 3412`, `CUM' KTA38`
- 자동으로 대문자로 변환됩니다.

**작업내용**: 
- 수행한 작업 설명
- 예시: `엔진 오버홀`, `정비`

**장소**: 
- 작업 장소
- 예시: `동하`, `공장`, `부산`

**작업자**: 
- 직급 + 이름
- 예시: `대리 하영광`, `차장 이주호`
- 기울임체: `*대리 하영광*` → 0.5공

**인원**: 
- **자동 계산**됩니다.
- 수정할 수 없습니다.

**동반자**: 
- 세 가지 형식 지원:
  1. **본사 직원**: `홍길동, 박명수, 김철수`
  2. **외주 도급**: `업체명(이름, 이름)` → 1공
  3. **외주 일당**: `업체명[이름, 이름]` → 인원수만큼
- 기울임체: `*홍길동*` → 0.5공

#### 인원 계산 규칙

**작업자**:
- 일반: 1공
- 기울임체 (`*이름*`): 0.5공

**동반자**:
- **본사 직원**: 각 1공
  - `홍길동, 박명수` → 2공
  - `*홍길동*, 박명수` → 1.5공
  
- **외주 도급**: 무조건 1공
  - `ABC업체(홍길동, 박명수)` → 1공
  - `ABC업체(홍길동, 박명수, 김철수, 이영희)` → 1공
  
- **외주 일당**: 인원수만큼
  - `ABC업체[홍길동]` → 1공
  - `ABC업체[홍길동, 박명수]` → 2공
  - `ABC업체[*홍길동*, 박명수]` → 1.5공

**혼합 입력**:
- `홍길동, 박명수, ABC업체(김철수, 이영희)` → 3공
  - 홍길동: 1공
  - 박명수: 1공
  - ABC업체 도급: 1공

- `조각상, 개인[김영언, 이정훈]` → 3공
  - 조각상: 1공
  - 개인 일당: 2공

### 2. 저장하기

1. 데이터 입력 완료 후 "저장" 버튼 클릭
2. "저장되었습니다" 메시지 확인
3. 인원이 자동으로 재계산됩니다

> ⚠️ **주의**: 저장하지 않으면 데이터가 사라집니다!

### 3. 어제 작업 불러오기

1. "어제 작업 불러오기" 버튼 클릭
2. 어제 데이터가 오늘로 복사됩니다
3. 필요한 부분만 수정하고 저장

## 고급 기능

### Excel 내보내기

1. 화면 상단의 "보고서" 버튼 클릭
2. "Excel 내보내기" 버튼 클릭
3. 파일은 **바탕화면**에 저장됩니다
4. 파일명: `작업현황_YYYY-MM-DD.xlsx`

### 클라우드 동기화

#### 자동 동기화
- 프로그램 시작 시: 클라우드 → 로컬
- 저장 시: 로컬 → 클라우드
- 프로그램 종료 시: 로컬 → 클라우드

#### 수동 동기화
1. 상단의 ☁️ 버튼 클릭
2. "클라우드로 동기화하시겠습니까?" 확인
3. "예" 클릭

#### 지원 클라우드
- Google Drive
- OneDrive
- Dropbox

> 💡 **팁**: 클라우드를 사용하면 출장 중에도 데이터에 접근할 수 있습니다.

### 보고서 보기

1. "보고서" 버튼 클릭
2. 요약 정보 확인:
   - 총 작업 건수
   - 총 투입 인원
   - 작성자
3. 상세 내역 확인

## 문제 해결

### Q: 프로그램이 실행되지 않아요

**A**: 
1. 작업 관리자에서 중복 실행 확인
2. 프로그램 재설치
3. IT 담당자에게 문의

### Q: 저장이 안돼요

**A**: 
1. 로그인 상태 확인
2. 프로그램 재시작
3. 데이터 폴더 권한 확인

### Q: 인원 계산이 이상해요

**A**: 
1. 동반자 입력 형식 확인:
   - 본사: `이름, 이름`
   - 도급: `업체명(이름, 이름)`
   - 일당: `업체명[이름, 이름]`
2. 쉼표로 정확히 구분했는지 확인
3. 괄호/대괄호 짝이 맞는지 확인

### Q: 클라우드 동기화가 안돼요

**A**: 
1. Google Drive / OneDrive 설치 확인
2. 클라우드 앱 실행 확인
3. 설정 파일 확인 (`config/settings.json`)

### Q: 데이터가 사라졌어요

**A**: 
1. `data/backups/` 폴더 확인
2. 클라우드에서 복구
3. IT 담당자에게 문의

### Q: 백신이 프로그램을 차단해요

**A**: 
1. Windows Defender 설정
2. "바이러스 및 위협 방지"
3. "제외 추가"
4. 프로그램 폴더 추가

## 팁과 요령

### 💡 효율적인 입력

**1. 비슷한 작업이 많을 때**:
- "어제 작업 불러오기" 사용
- 변경된 부분만 수정

**2. 기울임체 빠르게 입력**:
- `*이름*` 직접 입력
- 예: `*홍길동*`

**3. 외주사 입력**:
- 도급 (인원 무관): `업체명(이름들)`
- 일당 (인원 계산): `업체명[이름들]`

### 📝 입력 예시

**케이스 1: 본사 직원만**
```
작업자: 대리 하영광
동반자: 홍길동, 박명수, 김철수
→ 인원: 4공
```

**케이스 2: 본사 + 외주 도급**
```
작업자: 차장 이주호
동반자: 조각상, 아다한, 성진(정동진, 이덕윤)
→ 인원: 4공 (차장 1 + 조각상 1 + 아다한 1 + 성진 도급 1)
```

**케이스 3: 본사 + 외주 일당**
```
작업자: 대리 박명수
동반자: 홍길동, 개인[김영언, 이정훈]
→ 인원: 4공 (대리 1 + 홍길동 1 + 개인 일당 2)
```

**케이스 4: 기울임체 사용**
```
작업자: *대리 하영광*
동반자: *홍길동*, 박명수, 개인[김철수, *이영희*]
→ 인원: 3공 (0.5 + 0.5 + 1 + 1 + 0.5)
```
